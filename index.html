Projects

ECG Monitoring System

Our graduation project revolutionizes healthcare with an innovative ECG Monitoring System. A wearable device captures ECG signals, seamlessly transmitting them to a Flutter-based mobile app.

Key Features:

  • 📱 Real-Time Access: The mobile app provides secure, real-time access to stored ECG data in the cloud, enabling remote monitoring for doctors and patients.
  • 🏥 Hospital-Side Webpage: A user-friendly webpage allows for comprehensive patient monitoring and immediate alerts for abnormalities.
  • 🤖 AI-Powered Analysis: Integration of a machine learning model enables precise detection of abnormalities, triggering timely alerts to hospitals.
  • 💡 Comprehensive Ecosystem: Our system includes the wearable device, mobile app, hospital-side webpage (powered by Flask), and secure data storage (facilitated by Firebase).

Time Series Forecasting Model for Weather Data

Our final project in data science training with GSG, this endeavor involved building a Time Series forecasting model for weather data. Here's a glimpse of our remarkable journey

Key Features:

  • 📊 Exploratory Data Analysis (EDA): We began by selecting a comprehensive weather dataset and delved into it with EDA. This step helped us understand the data's patterns, distributions, and key features.
  • 🔧 Data Preprocessing: Meticulously handled missing values, outliers, and performed feature engineering to extract relevant information for accurate forecasting.
  • 🏗️ Model Building: Explored various algorithms, with Linear Regression and Stochastic Gradient Descent (SGD) standing out as the top performers.
  • 🧪 Model Evaluation: Rigorous testing and validation ensured reliability and generalization to new data, employing techniques like cross-validation.
  • 📱 Flutter Mobile App: Developed a user-friendly Flutter mobile application enabling users to access current weather temperature and a 5-day forecast, powered by our predictive model.
  • 🌐 Backend with Flask: Implemented Flask for authentication and deployed our model to make real-time predictions accessible through the app.
